# Expo shopping app

This is an demo shopping app rewritten based on [recraft-react-native-shopping-app](https://github.com/recraftrelic/recraft-react-native-shopping-app). Thanks `recraftrelic`

Libraries:
- react-navigation
- redux
- redux-actions
- react-native-carousel
